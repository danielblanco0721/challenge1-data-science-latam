# Análisis de Datos - Tiendas E-commerce

## 📌 Descripción
Proyecto del Challenge 1 de Data Science (Alura Latam). Análisis del desempeño de 4 tiendas e-commerce para recomendar la mejor opción de asociación comercial.

## 🛠️ Tecnologías
- Python 3.x
- pandas
- matplotlib
- Google Colab

## 📊 Datos
Cuatro archivos CSV con transacciones de tiendas que incluyen:
- Producto, categoría, precio, costo de envío
- Fecha, vendedor, ubicación, calificación
- Método de pago, coordenadas geográficas

## 📈 Análisis Realizados
1. **Ingresos totales** por tienda
2. **Ventas por categoría** (más y menos populares)
3. **Calificación promedio** de clientes
4. **Productos más/menos vendidos**
5. **Costo de envío promedio**
6. **Visualizaciones** (barras, circular, línea, dispersión)

## 🎯 Resultados Principales

| Tienda | Ingresos | Calificación | Envío Promedio |
|--------|----------|--------------|----------------|
| Tienda 1 | $1.150M | 3,977 | $26.019 |
| Tienda 2 | $1.116M | 4,037 | $25.216 |
| Tienda 3 | $1.098M | 4,048 | $24.806 |
| Tienda 4 | $1.038M | 3,996 | $23.459 |

**Categorías líderes:** Muebles, Electrónicos, Juguetes

## ✅ Recomendación
**Tienda 3** - Mejor combinación de:
- Mayor satisfacción del cliente (4,048/5,0)
- Liderazgo en categoría principal (Muebles: 499 ventas)
- Envíos competitivos
- Ingresos estables
